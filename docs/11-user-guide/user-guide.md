# Этап 11. Руководства

## Часть 1. Руководство пользователя (читателя)

### Регистрация и вход
1. Откройте приложение (`http://localhost:5173`).
2. «Регистрация» → укажите имя, email и пароль (не менее 6 символов) → «Зарегистрироваться».
3. «Вход» → введите email и пароль.

### Поиск и просмотр книг
- На главной странице отображается каталог по 12 книг на странице («Назад»/«Вперёд»).
- Введите запрос в строку поиска и нажмите «Искать» — поиск идёт по названию, описанию, автору,
  жанру (с учётом словоформ) и по ISBN (точно).
- «Подробнее» — карточка книги.

### Выдача и возврат
- В карточке доступной книги нажмите **«Взять читать»** — книга закрепится за вами.
- Вернуть книгу можно кнопкой **«Вернуть книгу»**. Вернуть может только тот, кто её взял.

## Часть 2. Руководство администратора (библиотекаря)

### Получение доступа
- Стартовый библиотекарь создаётся автоматически при первом запуске (настройки `app.admin.*`).
  По умолчанию: email и пароль из `application.properties` — **смените их**.

### Панель библиотекаря (`/admin`)
Доступна по кнопке «Панель библиотекаря» на главной (видна только библиотекарю).
- **Добавить книгу** — название, автор, ISBN, жанр, описание.
- **Удалить книгу по ISBN** — введите ISBN, подтвердите удаление.
- **Добавить библиотекаря** — имя, email, пароль (создаёт новую учётку с ролью LIBRARIAN).

### Развёртывание
Инструкции по установке и настройке — в [docs/10-deployment](../10-deployment/deployment.md).

### Типичные проблемы
| Симптом | Причина | Решение |
|---------|---------|---------|
| 401 при запросах | Сессия не создана/истекла | Войти заново |
| Поиск по автору не работает | Старый маппинг индекса | Пересоздать индекс `books` (см. deployment) |
| Книга «уже выдана» | Есть активная выдача | Дождаться возврата владельцем |
| Пустой каталог | Индекс не заполнен | Проверить `app.books.seed-enabled` и логи запуска |
